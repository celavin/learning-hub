# markdown语法快速学习
每个"#"代表一级标题
# 1.这是一级标题 (用于文章大标题)
## 这是二级标题 (用于章节)
### 这是三级标题 (用于小点)
# 2.待办事项
- [ ] 这是未办事项
- [x] 这是已办事项

# 3.代码块
用三个反引号包裹代码,记得写语言名称
``` python
    def hello():
        print("Hello,World")
```
``` java
    public class hello(){
        public static void main(){
            System.out.println("Hello,World!");
        }
    }
```
